# UI State Documentation

This folder contains detailed design documentation for each state of the MyPak interface.

## States

- **[healthy.md](./healthy.md)** - All products well-stocked (16+ weeks)
- **[single-urgent.md](./single-urgent.md)** - One container needs ordering soon
- **[multiple-urgent.md](./multiple-urgent.md)** - Multiple containers critical
- **[mixed.md](./mixed.md)** - Some urgent, some healthy

## Purpose

Each state document includes:
- Design philosophy
- Component-by-component breakdown
- Color palette
- Messaging guidelines
- User journey
- Implementation checklist
